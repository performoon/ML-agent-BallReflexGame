# ML-agent-BallReflexGame

このプロジェクトは、強化学習に関するプログラムです。
落ちてくるボールを跳ね返すボール反射ゲームを環境として強化学習を行っています。

## プロジェクトの概要

このプログラムは、強化学習を用いてゲームAIを実現するプログラムです。
